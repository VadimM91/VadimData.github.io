About This Project

Aiming to present my work in the most pleasent and convenient way, I found myself wavering between building a website and storing the projects on GitHub. After some consideration and self-reflection about my own goals and preferences, I decided on making a website. However, services like Wix and Webflow did not allow the level of customisation I was striving for. The end result is the self-made website you see here, which is based on a template and edited using HTML.

Working on this project made me appreaciate the fine work of web designers, which is often taken for granted when casually browsing the internet.
